c++题解2
==========

题意
-----
给定一个数组 nums，写一个函数将所有的 0 移动到数组的最后，同时保持非零元素的相对顺序。

**注意：**

- 不允许拷贝数组，必须在原位置进行。
- 操作次数最少。

例 ::

    给定 nums = [0, 1, 0, 3, 12]，在调用函数之后，nums 应该是 [1, 3, 12, 0, 0]。


解答思路
--------

* 遍历数组，将不为 0 的元素依次向前移动，记录下位置下标。

* 将该下标开始到余下的所有元素置 0。
