python题解¶

题意¶

从数组中查找两个元素值(值可以相同，但不能是一个位置的)，并且保证两者之和 等于目标值,返回这两个元素的坐标位置

解答思路¶

  • 遍历数组时为每个元素值建立hash索引关系，其中key值为该元素值距离目标值的 差值,value则为该元素值的位置,当后面的元素值有等于前面的目标差值时，则找到我们 想要的结果．

# -*- coding: utf-8 -*-


class Solution:
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """

        rels = {}
        for pos in range(len(nums)):
            if nums[pos] in rels:
                return [rels[nums[pos]], pos]
            else:
                rels[target-nums[pos]] = pos

        return []

s = Solution()
print(s.twoSum([2, 7, 11, 15], 9))
