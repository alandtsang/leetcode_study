c++题解2¶

题意¶

给定一个排序的数组和 target，如果找到 target，返回索引。

如果没找到，返回 target 应该被插入位置的索引。

例

[1,3,5,6], 5 → 2
[1,3,5,6], 2 → 1
[1,3,5,6], 7 → 4
[1,3,5,6], 0 → 0

解答思路¶

  • 使用 lower_bound 返回不比 target 小的第一个元素的迭代器。
  • 如果数组的所有元素都小于 target，则 lower_bound 返回最后一个迭代器。
  • 计算返回迭代器与数组起始迭代器之间的偏移即可。

class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        // An iterator to the lower bound of target in the range.
        auto low = std::lower_bound(nums.begin(), nums.end(), target);
        return low - nums.begin();
    }
};
