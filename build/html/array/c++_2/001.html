c++题解2¶

题意¶

给定一个整数数组，如果存在两个元素相加等于一个特定值，返回这两个元素的索引。

假设每个输入都只有一个解决方案，且不能使用相同的元素两次。

例

nums = [2, 7, 11, 15], target = 9

由于 nums[0] + nums[1] = 2 + 7 = 9，所以 [0, 1]。

解答思路¶

  • 第一次遍历数组，将数组的元素和索引作为 key/value 存到 unordered_map。
  • 第二次遍历数组，查找 target - i 的值是否在 unordered_map 中，如果在，将元素下标返回。

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        vector<int> ret;
        unordered_map<int, int> v;
        
        for (unsigned i = 0; i < nums.size(); i++)
            v.insert({nums[i], i});  // <key:element, value:index>
        
        for (auto it = nums.begin(); it != nums.end(); ++it) {
            auto it2 = v.find(target - *it);
            // the index in the vector can not be the same as in unordered_map
            if (it2 != v.end() && (it - nums.begin()) != it2->second) {
                ret.push_back(it - nums.begin());
                ret.push_back(it2->second);
                break;
            }
        }
        return ret;
    }
};
